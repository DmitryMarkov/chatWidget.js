https://dmitrymarkov.github.io/chatWidget.js/

### COMMANDS

#### run

After you have installed all dependencies you can now run the app with:
```bash
npm start
```

#### lint

```bash
npm run eslint
```

#### unit test

```bash
npm test
```

#### generate pug templates

```bash
npm run pug
```

#### generate sass templates

```bash
npm run sass
```
