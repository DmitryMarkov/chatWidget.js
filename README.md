## Commands


After you have installed all dependencies you can now run the app with:
```bash
npm start
```

lint project files
```bash
npm run eslint
```

run unit testing
```bash
npm test
```

generate pug templates
```bash
npm run pug
```

generate sass styles
```bash
npm run sass
```
